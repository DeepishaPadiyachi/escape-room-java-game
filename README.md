# Escape Room - Java Platformer Game

A 2D platformer game built with Java Swing featuring 
6 levels, physics-based movement, and puzzle mechanics.

## Features
- 6 handcrafted levels with increasing difficulty
- Physics engine with gravity and collision detection
- Moving saw enemy with rotation animation
- Key and door puzzle mechanics
- Level select screen with timer
- Character blinking animation

## Tech Used
Java, Java Swing, AWT

## How to Run
Compile and run CjGame.java in any Java IDE
